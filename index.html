Full page and multi scroll - A Nilead experiment

Introduction

This is a simple javascript/stylesheet plugin to create the full page and multi scroll effects.

This plugin was inspired by 2 modules on Alvaro Trigo
  1. Full page scroll plugin to create full page scrolling effect
  2. Multi scroll plugin to create split screen scrolling effect
We wanted to use both of these modules on one single web page which we were able to do by hacking around a bit. The result was rather hackish so we decided to create our own plugin that:
  1. Supports both full page and multi scroll
  2. Lightweight with the base essential features (no dependency on any library such as jQuery)
  3. Supports only modern browsers to keep things simple (old browsers such as IE9 and older may not work)
