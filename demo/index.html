Welcome to the demo page of our multi scroll experiment.
This is a experimental project of Nilead Website design and development platform and services.
This demo should show you how our multi scroll works on your broswer. Please note that the code is not optimized for older browsers such as IE9 and below. View the source code to see how you should structure your code for multi scroll, feel free to fork the library to add features, bug fixes and send Pull Requests.

Use your mouse to scroll down (or swipe down if you are on a touch device) to see the effect

The source of this multi scroll experiment is hosted on GitHub
